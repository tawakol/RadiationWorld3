Before running the test, please set up the Junit and easymock library path.
For easymock, objenesis, cglib:
1. if you use eclipse:
	a. right click the whole “Minecraft” project package inside of eclipse
	b. select “Properties”
	c. select “Java Build Path”
	e. select “Libraries”
	f. select “Add Exernal JARs” and select the “easymock-3.2-tests.jar”, 		“objenesis-2.1.jar”, and “cglib-3.1.jar” in the 
		src/test/java/group4/libraries folder
